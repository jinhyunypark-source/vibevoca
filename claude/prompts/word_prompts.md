# AI 단어 컨텐츠 관리 프롬프트 템플릿

## 1. 새 단어 생성

```
당신은 영어 어휘 학습 전문가입니다. 다음 카테고리와 데크에 맞는 새로운 영어 단어를 생성해주세요.

카테고리: {category}
데크: {deck}
기존 단어들: {existing_words}

요구사항:
1. 기존 단어와 중복되지 않는 새 단어 {count}개 생성
2. 데크 주제에 맞는 단어 선택
3. 각 단어에 대해 다음 정보 제공:
   - front_text: 영어 단어/표현
   - back_text: 한글 뜻
   - example_sentences: 예문 2개 (배열)

JSON 형식으로 출력:
[
  {
    "front_text": "word",
    "back_text": "뜻",
    "example_sentences": ["예문1", "예문2"]
  }
]
```

## 2. 예문 생성 (컨텍스트 기반)

```
당신은 영어 문장 작성 전문가입니다. 주어진 단어와 컨텍스트에 맞는 자연스러운 예문을 생성해주세요.

단어: {word}
의미: {meaning}
컨텍스트:
  - 장소: {place}
  - 감정: {emotion}
  - 환경: {environment}

요구사항:
1. 컨텍스트에 맞는 자연스러운 예문 1개 생성
2. 일상적인 상황에서 사용할 수 있는 문장
3. 단어의 의미가 명확히 드러나는 문장

JSON 형식으로 출력:
{
  "sentence": "영어 예문",
  "translation": "한글 번역"
}
```

## 3. 단어 설명 개선

```
당신은 영어 교육 전문가입니다. 다음 단어의 설명을 더 이해하기 쉽게 개선해주세요.

단어: {word}
현재 설명: {current_definition}

요구사항:
1. 간결하고 명확한 한글 설명
2. 핵심 의미를 잘 전달
3. 학습자가 쉽게 기억할 수 있는 표현

출력:
{
  "improved_definition": "개선된 설명"
}
```

## 4. 유사 단어 추천

```
당신은 영어 어휘 전문가입니다. 주어진 단어와 관련된 유사 단어를 추천해주세요.

기준 단어: {word}
의미: {meaning}
데크: {deck}

요구사항:
1. 의미가 유사하거나 관련된 단어 {count}개 추천
2. 각 단어의 뜻과 차이점 설명
3. 학습에 도움이 되는 단어 선택

JSON 형식으로 출력:
[
  {
    "word": "유사 단어",
    "meaning": "뜻",
    "difference": "기준 단어와의 차이점"
  }
]
```

## 5. 데크 주제에 맞는 단어 검증

```
당신은 영어 어휘 분류 전문가입니다. 주어진 단어가 데크 주제에 적합한지 검증해주세요.

단어: {word}
의미: {meaning}
데크: {deck}
데크 설명: {deck_description}

평가 기준:
1. 단어가 데크 주제와 관련이 있는가?
2. 난이도가 데크의 다른 단어와 비슷한가?
3. 학습 가치가 있는가?

JSON 형식으로 출력:
{
  "is_suitable": true/false,
  "reason": "판단 이유",
  "suggestion": "부적합한 경우 대체 추천"
}
```
