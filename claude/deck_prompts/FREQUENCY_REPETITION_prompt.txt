# 영어 학습 예문 생성 요청

## 단어 목록
1. Intermittent - 간헐적인 (멈췄다 시작했다 하는)
2. Sporadic - 산발적인 (불규칙하게 드문드문 일어나는)
3. Occasional - 가끔의 (자주는 아니지만 때때로)
4. Periodic - 주기적인 (일정한 간격을 두고 일어나는)
5. Recurrent - 재발하는, 되풀이되는 (자꾸 다시 일어나는)
6. Frequent - 빈번한 (자주 일어나는)
7. Constant - 끊임없는 (변함없이 계속되는)
8. Continual - 빈번한 (짜증 날 정도로 자꾸 반복되는)
9. Regular - 규칙적인 (정해진 패턴대로)
10. Irregular - 불규칙한 (패턴이 없는)
11. Cyclical - 순환하는 (주기를 돌며 반복되는)
12. Hourly - 매 시간의 (1시간마다)
13. Biennial - 2년마다의 (격년의)
14. Perennial - 다년생의, 연중 계속되는 (오랫동안 반복되는)
15. Habitual - 습관적인 (버릇처럼 하는)
16. Customary - 관례적인 (관습에 따라 늘 하는)
17. Routine - 일상적인 (정해진 순서대로 하는)
18. Systematic - 체계적인 (계획에 따라 규칙적으로)
19. Alternate - 번갈아 일어나는 (하나 건너 하나)
20. Consecutive - 연속적인 (중단 없이 연달아)
21. Successive - 잇따른 (순서대로 이어지는)
22. Spasmodic - 발작적인 (갑자기 확 일어났다 멈추는)
23. Fitful - 자다 깨다 하는, 불규칙한 (잠깐씩 끊기는)
24. Rare - 드문 (자주 일어나지 않는)
25. Infrequent - 드문 (빈도가 낮은)
26. Seldom - 좀처럼 ~않는 (거의 없는)
27. Scarce - 희귀한, 부족한 (찾아보기 힘든)
28. Commonplace - 흔한 (너무 자주 있어서 평범한)
29. Ubiquitous - 어디에나 있는 (동시에 도처에 존재하는)
30. Annual - 매년의 (1년마다)

## 작업 요구사항

당신은 영어 학습 예문을 생성하는 전문가입니다. 위의 단어들에 대해 학습자가 쉽게 이해하고 기억할 수 있는 자연스러운 예문을 만들어주세요.

### 예문 생성 가이드라인

1. **예문 개수**: 각 단어당 5-8개의 예문 생성
2. **예문 길이**: 10-20단어 정도의 자연스러운 문장
3. **태그 활용**:
   - 위의 추천 태그(상황/맥락)를 자연스럽게 활용
   - 각 예문마다 어울리는 태그 5개 내외만 선택
   - **중요**: 모든 태그를 억지로 사용하지 말 것
4. **다양성**: 다양한 맥락과 상황을 다룰 것
5. **실용성**: 실생활에서 사용 가능한 자연스러운 문장
6. **한국어 번역**:
   - 직역이 아닌 **자연스러운 한국어 의역**
   - 한국인이 실제로 사용하는 표현으로 번역

### 출력 형식

반드시 아래의 JSON 형식으로 출력해주세요:

```json
{
  "deck_title": "FREQUENCY_REPETITION",
  "total_sentences": 0,
  "sentences": [
    {
      "word": "단어",
      "sentence_en": "영어 예문",
      "sentence_ko": "자연스러운 한국어 번역",
      "tags": ["선택된", "태그", "5개", "내외"]
    }
  ]
}
```
