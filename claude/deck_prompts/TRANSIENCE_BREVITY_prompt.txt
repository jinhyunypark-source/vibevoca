## 단어 목록
1. Ephemeral - 덧없는, 단명하는 (하루살이처럼 수명이 짧은)
2. Fleeting - 순식간의, 덧없는 (빠르게 지나가 버리는)
3. Interim - 중간의, 임시의 (다음 단계 전까지의)
4. Provisional - 임시의, 잠정적인 (나중에 바뀔 수 있는)
5. Transient - 일시적인, 머물다 가는 (오래 머물지 않는)
6. Transitory - 일시적인 (영원하지 않고 지나가는 성질의)
7. Temporary - 임시의 (정해진 기간만 쓰는)
8. Fugitive - 도망 다니는, 덧없는 (붙잡아 둘 수 없는)
9. Momentary - 순간적인 (아주 짧은 찰나의)
10. Brief - 짧은, 잠시의 (시간이 길지 않은)
11. Short-lived - 단명하는, 오래가지 못하는
12. Evanescent - 쉬이 사라지는, 덧없는 (안개처럼 사라지는)
13. Passing - 지나가는, 일시적인 (머물지 않는)
14. Tentative - 잠정적인, 머뭇거리는 (확정되지 않은)
15. Cursory - 대충 하는, 피상적인 (시간을 들이지 않은)
16. Instantaneous - 즉각적인 (동시에 일어나는)
17. Abrupt - 갑작스러운 (예고 없이 뚝 끊기는)
18. Sudden - 갑작스러운 (예상치 못한)
19. Short-term - 단기적인 (가까운 미래만 보는)
20. Impermanent - 영구적이지 않은 (변하기 쉬운)
21. Volatile - 휘발성의, 변덕스러운 (금방 변해버리는)
22. Swift - 신속한 (재빠르게 지나가는)
23. Hasty - 서두르는, 성급한 (시간을 충분히 안 쓴)
24. Fly-by-night - 믿을 수 없는, 덧없는 (야반도주하는/잠깐 하고 마는)
25. Mortal - 언젠가 죽는, 필멸의 (유한한 생명의)
26. Finite - 유한한 (한계가 있는)
27. Phase - 단계, 시기 (지나가는 한 국면)
28. Spell - 잠깐의 기간 (날씨나 활동의 짧은 기간)
29. Flash - 섬광, 순간 (번쩍하는 사이)
30. Blink - 눈 깜짝할 사이 (아주 짧은 순간)

## 작업 요구사항

당신은 영어 학습 예문을 생성하는 전문가입니다. 위의 단어들에 대해 학습자가 쉽게 이해하고 기억할 수 있는 자연스러운 예문을 만들어주세요.

### 예문 생성 가이드라인

1. **예문 개수**: 각 단어당 5-8개의 예문 생성
2. **예문 길이**: 10-20단어 정도의 자연스러운 문장
3. **태그 활용**:
   - 위의 추천 태그(상황/맥락)를 자연스럽게 활용
   - 각 예문마다 어울리는 태그 5개 내외만 선택
   - **중요**: 모든 태그를 억지로 사용하지 말 것
4. **다양성**: 다양한 맥락과 상황을 다룰 것
5. **실용성**: 실생활에서 사용 가능한 자연스러운 문장
6. **한국어 번역**:
   - 직역이 아닌 **자연스러운 한국어 의역**
   - 한국인이 실제로 사용하는 표현으로 번역
