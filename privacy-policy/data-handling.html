데이터 처리 규정

VibeVoca(이하 "서비스")는 사용자의 데이터를 투명하게 처리하며, 사용자의 권리를 최우선으로 보장합니다.

📌 계정 삭제 요청 방법

사용자는 언제든지 계정 삭제를 요청할 수 있으며, 다음의 방법으로 진행할 수 있습니다:

  1. 앱 내 설정: VibeVoca 앱 내 설정 메뉴에서 '계정 삭제' 버튼을 통해 즉시 삭제
  2. 이메일 요청: 아래 이메일로 계정 삭제 요청 (24시간 내 처리)

계정 삭제 문의

이메일 제목: "계정 삭제 요청" / 본문: 가입 이메일 또는 사용자 ID

1. 수집 및 처리되는 데이터

VibeVoca는 다음과 같은 데이터를 수집 및 처리합니다:

  • 계정 정보: 이메일 주소, 로그인 식별자 (인증 및 계정 관리 목적)
  • 학습 데이터: 사용자가 생성한 학습 기록, 진행 상황, 즐겨찾기 (서비스 제공 및 개인화 목적)
  • 기기 정보: 앱 버전, OS 버전, 기기 ID (서비스 최적화 및 오류 추적 목적)

2. 계정 삭제 시 데이터 처리

사용자가 계정 삭제를 요청하면 다음과 같이 처리됩니다:

데이터 유형 처리 방법 보관 기간
계정 정보 (이메일, 로그인 ID) 즉시 삭제 삭제 요청 후 즉시
학습 데이터 (진행 상황, 기록) 즉시 삭제 삭제 요청 후 즉시
로그 데이터 (오류 기록, 접속 기록) 익명화 후 보관 최대 30일
법적 의무 이행 데이터* 최소 정보만 보관 관련 법령에 따라 최대 1년

* 부정 사용 방지, 사기 방지 등 법적 의무 이행을 위해 필요한 경우에 한해 최소한의 정보(예: 거래 기록)만 보관될 수 있습니다.

3. 데이터 보관 정책

  • 활성 계정: 서비스 이용 기간 동안 안전하게 보관
  • 비활성 계정: 1년 이상 미사용 시 사전 안내 후 자동 삭제
  • 삭제 요청 계정: 요청 즉시 또는 24시간 이내 삭제 완료

4. 사용자 권리 및 보장

VibeVoca 사용자는 다음의 권리를 가집니다:

  • 열람권: 본인의 개인정보 및 데이터를 언제든지 확인 가능
  • 정정권: 잘못된 정보를 수정 요청 가능
  • 삭제권: 언제든지 계정 및 데이터 삭제 요청 가능
  • 처리 정지권: 개인정보 처리의 일시 정지 요청 가능
  • 이동권: 본인의 데이터를 다운로드하여 이동 가능

5. 제3자 제공 및 공유

VibeVoca는 사용자의 개인정보를 어떠한 제3자에게도 판매, 공유 또는 제공하지 않습니다.

다만, 다음의 경우에 한해 최소한의 정보가 공유될 수 있습니다:

  • 법적 요구사항 준수 (법원 명령, 수사기관 요청 등)
  • 사용자의 명시적 동의가 있는 경우

6. 데이터 보안

VibeVoca는 사용자 데이터를 안전하게 보호하기 위해 다음의 조치를 취합니다:

  • 암호화된 통신 (HTTPS/TLS)
  • 데이터베이스 암호화 저장
  • 정기적인 보안 점검 및 업데이트
  • 접근 권한 엄격 관리
